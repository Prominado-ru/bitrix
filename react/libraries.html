# Библиотеки

  • Expo SDK - Много разных кроссплатформенных полезностей
  • react-native-datepicker – Datepicker (iOS / Android)
  • react-native-dropdownalert – Вывод сообщений в статусбаре (iOS / Android)
  • react-native-elements – Подборка полезных компонентов (списки, иконки, итд) (iOS / Android)
  • react-native-keyboard-aware-scroll-view – В формах при фокусе на инпут подскроливает страницу (чтобы клавиатура не перекрывала инпут) (iOS / Android)
  • react-native-masked-text – Маска ввода для инпута (iOS / Android)
  • react-native-segmented-control-tab – Segment Controller (iOS / Android)
  • react-native-side-menu – Боковое свайп-меню (iOS / Android)
  • react-native-swipe-list-view – ListView с возможностью свайпа строки (iOS / Android)
  • react-native-swiper – Onboarding (iOS / Android)
  • react-native-text-input-mask – Маска ввода для инпута (iOS / Android)
  • react-navigation – Навигация (iOS / Android)
