# Установка ПО:

  • Node.js – без него никак ¯_(ツ)_/¯
  • nmp – менеджер пакетов, практически все ставится им
  • Expo XDE – Надстройка над React Native
  • exp – Expo CLI
  • Homebrew – Установщик инструментов, которых нет в MacOS
  • watchman – Отслеживает измененные файлы
